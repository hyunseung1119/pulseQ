# PulseQ — 프론트엔드 화면 설계서

## 기술 스택
| 기술 | 용도 |
|------|------|
| React 18 + TypeScript | SPA 프레임워크 |
| Vite | 빌드 도구 |
| TanStack Query v5 | 서버 상태 관리 (캐싱, 리패칭) |
| TanStack Router | 타입 안전 라우팅 |
| Tailwind CSS v4 | 스타일링 |
| Recharts | 차트/그래프 |
| shadcn/ui | UI 컴포넌트 라이브러리 |

## 라우트 구조

```
/                           → 랜딩 페이지
/login                      → 로그인
/signup                     → 회원가입
/dashboard                  → 대시보드 메인
/dashboard/events           → 이벤트 목록
/dashboard/events/new       → 이벤트 생성
/dashboard/events/:id       → 이벤트 상세 + 실시간 모니터링
/dashboard/events/:id/edit  → 이벤트 수정
/dashboard/api-keys         → API 키 관리
/dashboard/usage            → 사용량/과금
/dashboard/settings         → 설정

# 대기열 사용자 화면 (테넌트 고객사의 end-user용)
/queue/:eventSlug           → 대기열 대기 화면
```
